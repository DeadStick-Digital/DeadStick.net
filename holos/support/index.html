Support

Use this page for Holos Document Vault support, privacy questions, account deletion guidance, forgotten PIN guidance, sync recovery, release evidence handling, and incident support.

Support contact: use the DeadStick Digital message form. Include the app build, device model, operating system version, active storage provider, and the shortest safe description of the issue.

Support paths

  • Data deletion - local vault data, cloud provider copies, exports, and subscription account records.
  • Forgotten PIN - what can and cannot be recovered when a local app PIN is lost.
  • Sync recovery - safe steps for provider migration, stale sync state, missing assets, or duplicate documents.
  • Evidence retention - how support and release QA should handle redacted evidence.
  • Incident support - how privacy, security, provider, account, and release escalations are triaged.

What not to send

Do not send PINs, passwords, one-time codes, OAuth tokens, private keys, RevenueCat keys, Supabase keys, App Store Connect keys, raw email addresses, Apple IDs, Google account identifiers, provider object ids, local file paths, document hashes, receipts, transaction ids, purchase tokens, billing records, document images, OCR text, translations, filenames, folders, tags, packets, generated PDFs, identity documents, financial documents, or medical documents unless support specifically asks for a narrowly redacted sample.

Helpful safe details

Useful support details include platform, app build, OS version, device model, active provider, visible redacted error label, last safe action taken, whether another unlocked device still has the expected vault, and whether a recent Holos Document Vault packet backup exists.
