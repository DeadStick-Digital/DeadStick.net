Holos Document Vault

Forgotten PIN

Holos Document Vault PINs are device-local app unlock credentials. They are not recoverable, and support cannot bypass or clear a stored PIN to open the same local vault.

What to try first

If Face ID or Touch ID is enabled and still works on the device, try biometric unlock first. If the PIN cannot be remembered and biometric unlock is unavailable or fails, the locked local vault cannot be opened on that device.

Recovery options

Recovery means restoring from another unlocked synced device, iCloud, Google Drive, or a previously exported Holos Document Vault packet backup after removing the locked local app data. Removing local app data can erase local-only documents, so confirm a good copy or backup before choosing that path.

Support boundary

DeadStick Digital support should not request, store, or handle user PINs. The app's forgotten PIN guidance keeps protected actions locked and does not clear the stored PIN credential.

Need help?

Use Holos Document Vault support if you need help identifying a synced device, provider copy, or packet backup before removing local app data.
